# ChainLedger Platform – Frontend

React + TypeScript + Vite SPA für den ChainLedger-Datenimport-Agenten.
Setup, Architektur und Beispielaufruf: siehe [`../README.de.md`](../README.de.md)
bzw. [`../README.md`](../README.md) im Projekt-Root, Abschnitt „Web-Frontend
(Phase 2)“.

## Quickstart

```bash
npm install
cp .env.example .env   # VITE_API_BASE_URL, Default: http://localhost:8000
npm run dev
```

Voraussetzung: Das Backend (`api/`) muss separat laufen (siehe Root-README).
